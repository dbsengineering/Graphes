## Menu

<img src="https://user-images.githubusercontent.com/8668325/34890399-063f0b38-f7d2-11e7-91b0-638dea9858de.png" width="400" height="790">

in menu, you can : 

* Reset graphic interface
* Erase nodes
* Erase arcs
* Add & edit nodes ( color, size, rename, delete)
* Add & edit arcs ( color, size, size Text, rename, delete)
* Move nodes & arcs
* Save Graph in file (.gra)
* Open .gra files
* Share graph
* Play to game :-)
